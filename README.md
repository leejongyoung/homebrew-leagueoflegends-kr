## Dependency tool
[Homebrew](https://brew.sh)

## Install
``` sh
brew tap leejongyoung/leagueoflegends-kr
brew install --cask league-of-legends-kr
```

## Uninstall
``` sh
brew uninstall --cask --zap league-of-legends-kr
```
